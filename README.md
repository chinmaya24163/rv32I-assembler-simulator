# \# RISC-V Assembler and Simulator

# 

# This project provides a complete workflow for assembling and simulating RISC-V (RV32I) programs.

# 

# \## Components

# 

# \* \*\*Assembler\*\*: Converts RISC-V assembly code into binary machine instructions

# \* \*\*Simulator\*\*: Executes the generated binary and outputs register and memory states



# \## Repositories

# 

# \* \*\*Assembler\*\*: \[custom-RV32I-assembler](https://github.com/chinmaya24163/custom-RV32I-assembler)

# \* \*\*Simulator\*\*: \[RV32I\_Simulator](https://github.com/chinmaya24163/RV32I\_Simulator)



# 

# \## Workflow

# 

# 1\. Write a RISC-V assembly program

# 2\. Assemble it into binary

# 3\. Run the binary in the simulator

# 

# \## Features

# 

# \* Supports custom RV32I instruction set

# \* Custom-built assembler

# \* Step-by-step simulation

# \* Displays register and memory updates
